myFlix-Angular

This was a one-person project in which I developed a frontend using Angular for the REST API. This API was previously utilized to connect 'myFlix-React' with the MongoDB database. The application is designed to facilitate user registration, login, display movies with descriptions and other relevant data from the database, allow users to save movies to their favorites list, and offer options for account editing and deletion.

Meet App

The application is designed to display nearby events to users in their selected cities. It was a one-person full-stack project, utilizing React on the frontend, AWS Lambda as the serverless backend API, and retrieving data through the Google Calendar API. The development process followed a Test-Driven and Behavior-Driven approach.

Website, GitHub

Additionally, a search bar has been implemented to allow users to search for a specific Pokémon by its name.
